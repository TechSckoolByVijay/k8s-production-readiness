# KEDA Exercises

This folder contains KEDA-related Kubernetes manifests used in the production readiness exercises.

## Files
- `00-chaos-generator.yaml`: Chaos generator manifest.
- `01-hpa-failure.yaml`: HPA failure scenario manifest.
- `02-keda-scaledobject.yaml`: KEDA ScaledObject example.
- `03-keda-production.yaml`: Production-ready KEDA configuration.
